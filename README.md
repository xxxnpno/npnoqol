## Features

### Hypixel in General
- **Auto GG**  
  Automatically sends "good game" at the end of a match with random capitalization.  
  *(Currently supported only for certain gamemodes.)*

- **Nick Manager**  
  Alerts you when a nicked player joins your lobby.  
  *(Currently supported only for certain gamemodes.)*

- **Chat Manager**
  Filters your chat
  
### Commands
- **/sethypixelapikey (key)**  
  Set your Hypixel API key to enable features.

### Supported Gamemodes
- **Blitz Survival Games**  
  - Deobfuscates chat join messages  
  - Deobfuscates nametags  
  - Deobfuscates tab list  
  - Adds a stats overlay in the tab (overall wins & overall KDR)  
  - Displays HP in tab & nametag  
  - Assigns unique colors for each team in team modes
